Board Game Resources

Board Game Cafes

One great way to try out different board games and see if you enjoy them before investing your money in purchasing them is to visit a board game cafe, of which there are several in NYC. The two most well-known are:

  • The Uncommons (Greenwich Village)
  • Hex & Co (three locations, listed below)
    • West Side
    • Upper East Side
    • Union Square

These cafes also have many board games for sale, so you have the option of purchasing a game as well!

Doing Board Game Research

My favorite online source of information about board games is BoardGameGeek, a website with descriptions of thousands of board games, as well as community rating/review system so that you can see how much other people enjoy the game and how difficult it is perceived to be.
